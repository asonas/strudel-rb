RubyKaigi 2026ローカルオーガナイザーのikarugaから、函館市内の街頭音声広告のBGM制作を依頼された。

**Why:** 昨年のgroovebox-ruby発表を見て、strudel-rbでBGMを作れると考えた。発表のデモとして使い「実はこれが広告BGMとして使われている」というトーク展開も可能。

**How to apply:**
- 30秒前後の落ち着いたシンプルなBGM
- 音声メッセージ(まつもとゆきひろさんなどが喋る予定)と合成される前提
- 締切: 2026-04-10(広告会社送付は4/16)
- 流れる場所: 函館駅前、五稜郭本町交差点
- 流れる時間帯: Day1-Day3の朝・夕方、Day4以降の日中
